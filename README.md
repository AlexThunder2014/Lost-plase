# LostPlaces Explorer NRW

Ein kostenloses Open-Source-Projekt zum Entdecken von Lost Places in Nordrhein-Westfalen.

## Technologien

- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- JSON

## Funktionen

- Interaktive Karte
- Suche
- Filter
- Kategorien
- Favoriten
- Detailseiten
- Statistiken
- Dark Mode
- Responsive Design

## Sicherheit

Das Projekt stellt ausschließlich öffentlich verfügbare Informationen dar und kennzeichnet den Zugangsstatus jedes Ortes deutlich. Es enthält keine Anleitungen zum unerlaubten Betreten privater oder gesperrter Gelände.
